# replaceIGmedia

Drop downloaded Instagram videos here using the filenames in the checklist below.
Once a video is dropped, gallery.html will use the local mp4 instead of the IG iframe embed (cleaner, faster, no IG branding).

## How to download from Instagram
1. Open the IG post / reel in a browser
2. Use any IG video downloader (snapinsta.app, igram.io, etc.) — paste the URL, download the mp4
3. Rename the file using the target filename below
4. Drop it in this folder
5. Tell Claude "videos are dropped" and Claude swaps the gallery to use them
